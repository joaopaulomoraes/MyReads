# MyReads App

A Book Lending App with React

## Getting Started

### Installing

Cloning the project.
```
https://github.com/joaopaulomoraes/MyReads.git myreads-app
```

Access the project directory.
```
cd myreads-app
```

Install dependencies.
```
yarn install
```

Serve with hot reload at http://localhost:3000.
```
yarn start
```
